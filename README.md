# generator-sda

> A generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator:

```bash
git clone git@github.com:sebastiendavid/generator-sda.git
cd generator-sda
npm install
sudo npm link
```

- Run: `yo sda`
